# 文章分析大师 (Article Analysis Master)

一个专业的文章深度分析技能，支持论证性和非论证性文章的系统化分析。

## 功能特点

### 📊 智能文章类型识别
- 自动识别文章类型（学术论文/评论文章/新闻报道/叙事散文等）
- 根据类型选择最合适的分析框架
- 论证性文章使用图尔敏模型深度剖析
- 非论证性文章采用叙事结构分析

### 🎯 十维度全面分析
1. **基本信息** - 标题、作者、发表信息、关键词、摘要、核心结论
2. **结构化总结** - 动态识别2-5个主要议题，详细展开
3. **核心观点/主题** - 提炼关键论点、独特洞见、挑战性观点
4. **论证/叙事分析** - 图尔敏模型六要素拆解（论证性）或叙事结构分析（非论证性）
5. **费曼学习法解析** - 用生活类比向12岁学生解释核心思想
6. **学习资源** - 区分文中引用资源和推荐资源
7. **信息可视化建议** - 提供2-3个可视化方案（图表类型、数据维度、工具推荐）
8. **Q&A** - 3-5个问题，覆盖理解性、应用性、批判性、拓展性
9. **写作风格分析** - 从6个维度分析（引言策略、结构组织、衔接技巧等）

### 🔍 严格的质量保证
- **输入验证** - 检查字数（≥100字）、格式、完整性
- **原文为本** - 只依据原文内容，无法验证的内容标注"原文未提及"
- **批判性思维** - 识别未明言的假设和潜在逻辑弱点
- **规范化输出** - 人名、引用、格式严格遵循规范

## 技术特点

### 图尔敏论证模型 (Toulmin Model)
对论证性文章使用图尔敏模型六要素：
1. **主张 (Claim)** - 作者的核心观点
2. **数据 (Data)** - 支撑主张的证据
3. **论证理由 (Warrant)** - 连接数据与主张的推理
4. **支撑 (Backing)** - 为论证理由提供的理论支持
5. **限定 (Qualifier)** - 主张的适用范围限定
6. **反驳 (Rebuttal)** - 预见和回应的反对意见

### 费曼学习法
- 用简单语言解释复杂概念
- 生活化类比和具体场景
- 分步骤说明逻辑链条
- 用反例验证理解深度

### 批判性阅读
- 识别隐含假设
- 评估论证强度
- 指出逻辑薄弱点
- 探讨潜在局限性
