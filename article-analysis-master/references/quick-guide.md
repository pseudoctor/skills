# 文章分析大师 - 快速参考指南

## 🚀 快速开始

### 一句话总结
智能识别文章类型，自动选择最合适的分析框架，输出十维度深度分析报告。

### 基本用法
```
/article-analysis-master
[粘贴你的文章]
```

## 📊 输出结构速览

### 论证性文章（10个部分）

```
零、文章类型判断
一、基本信息 (标题/作者/发表信息/关键词/摘要/核心结论)
二、结构化总结 (2-5个主要议题)
三、核心观点和关键洞见 (论点/洞见/挑战性观点)
四、图尔敏模型论证分析 (主张/数据/论证理由/支撑/限定/反驳)
五、费曼学习法解析 (生活类比/逻辑链条/验证理解)
六、学习资源 (文中引用+推荐资源)
七、信息可视化建议 (2-3个可视化方案)
八、Q&A (3-5个问题，多层次)
九、写作风格分析 (6个维度)
```

```
六、学习资源
七、信息可视化建议
八、Q&A
九、写作风格分析
```

## 🎯 关键要素详解

### 图尔敏模型六要素（论证性文章）

| 要素 | 英文 | 说明 | 示例 |
|-----|------|-----|------|
| **主张** | Claim | 作者想证明的观点 | "远程工作能提高生产力" |
| **数据** | Data | 支撑主张的证据 | "研究显示远程员工效率提高20%" |
| **论证理由** | Warrant | 连接数据与主张的推理 | "减少通勤时间让员工更专注工作" |
| **支撑** | Backing | 为论证理由提供的理论支持 | "时间管理理论证明连续工作块更高效" |
| **限定** | Qualifier | 主张的适用范围 | "对于自律性强的员工" |
| **反驳** | Rebuttal | 预见的反对意见及回应 | "虽然有人担心沟通效率..." |

### 费曼学习法三步骤

1. **核心概念简化** - 用12岁学生能懂的语言和生活类比
2. **逻辑链条分步说明** - 分解为3-5个简单步骤
3. **验证理解** - 用反例或边界案例测试

### Q&A 五种问题类型

| 类型 | 认知层次 | 示例 |
|-----|---------|------|
| 理解性 | 记忆/理解 | "文章的核心论点是什么？" |
| 理解性 | 理解/分析 | "作者如何论证X观点？" |
| 应用性 | 应用 | "这个理论如何应用到Y场景？" |
| 批判性 | 分析/评价 | "这个论证有哪些局限性？" |
| 拓展性 | 综合/创造 | "这与Z理论有何联系？"（选填） |

### 写作风格分析六维度

1. **引言策略** - 如何开篇吸引读者
2. **结构组织** - 整体结构特点
3. **衔接技巧** - 段落过渡方式
4. **情感基调** - 客观/激昂/沉思等
5. **语言特色** - 用词风格、句式特点、修辞手法
6. **说服策略** - 逻辑/情感/信誉的平衡

## ⚠️ 输入要求

### ✅ 符合要求
- 字数 ≥ 100字
- 包含段落结构的文字内容
- 完整的文章（不是仅摘要或残缺版本）

### ❌ 不符合要求
- ❌ 少于100字的短文
- ❌ 纯数据表格（无文字叙述）
- ❌ 仅标题和摘要
- ❌ 明显残缺的文章片段

## 💡 最佳实践

### 提高分析质量的技巧

1. **提供完整文章** - 越完整的文章，分析越准确
2. **包含元信息** - 如有标题、作者、发表信息，一并提供
3. **明确分析需求** - 如果有特定关注点，可以额外说明
4. **多轮对话** - 可以针对某个部分要求深入分析

## 🔧 核心约束

### 原文为本
- ✅ 所有分析基于原文内容
- ✅ 无法验证的内容标注"原文未提及"
- ⚠️ 唯一例外：学习资源部分可推荐外部资源（会明确标注）

### 批判性思维
- 识别未明言的假设
- 指出论证的潜在弱点
- 不做空泛的评价
- 公正评估贡献与局限

### 格式规范
- 人名首次出现给出中文和原文
- 结构化总结使用三级标题（###）
- 关键句使用加粗
- 全部输出使用简体中文

## 🆘 常见问题

**Q: 可以分析英文文章吗？**
A: 可以。系统会自动翻译为中文输出，关键术语会保留英文原文。

**Q: 诗歌、小说可以分析吗？**
A: 可以。会自动识别为非论证性文本，采用叙事结构分析框架。

**Q: 最短需要多少字？**
A: 至少100字。太短的文章无法进行深度分析。

**Q: 可以只分析某个部分吗？**
A: 可以在提交文章后，要求深入某个特定部分（如"请详细展开图尔敏模型分析"）。

**Q: 分析结果可以导出吗？**
A: 输出为Markdown格式，可以直接复制到任何支持Markdown的编辑器。
