# 输出格式模板

本文档包含各输出格式的详细模板和正文抽取模板。

---

## 格式A: Markdown 结构化报告

### 完整报告模板

```markdown
# [领域名称] 领域地图

## 1. 核心论文层级

### 必读论文
1. **[作者 年] [标题]**
   - 入选理由：[高被引/奠基性/代表XX派别]
   - 关键贡献：[一句话]

### 选读论文
1. **[作者 年] [标题]**
   - 入选理由：[XX派别的改进/特定应用]

### 了解论文
1. **[作者 年] [标题]**
   - 入选理由：[背景/辅助理解]

## 2. 派别划分

### 派别1: [名称]
- **边界定义**：[描述]
- **核心创意**：[一句话]
- **代表论文**：
  - [论文1]
  - [论文2]
- **优点**：[列点]
- **缺点**：[列点]

### 派别2: [名称]
...

## 3. 时间线与派别演化

### 派别1 演化脉络
[起点] → [关键节点] → [近期趋势]

### 派别间批评关系
| 批评者 | 被批评者 | 批评内容 |
|--------|----------|----------|
| 派别A | 派别B | [内容] |

## 4. 核心收敛

### 核心论文（约20篇）
[列表，含保留理由]

### 深挖派别选择
**选择**: [派别1, 派别2]
**理由**: [与子题匹配度/成熟度/突破点/学习成本]

### 二次检索式
```
([关键词1 OR 关键词2] AND 关键词3) NOT 关键词4
```

## 5. 评价维度

### 领域关键问题
- Q1: [问题]
- Q2: [问题]

### 评价维度
| 维度 | 说明 |
|------|------|
| 维度1 | [说明] |

### 场景权重
| 场景 | 高权重维度 |
|------|------------|
| 场景1 | 维度A, 维度B |

## 6. 深读计划

### 阅读顺序
1. [论文1] - 理由：[入门/综述]
2. [论文2] - 理由：[核心文献]
...

### 每篇抽取计划
[按论文列出，使用抽取模板]

## 7. 每派总结

### 派别1 总结
- **发展脉络**：[描述]
- **理论依据**：[描述]
- **主要成果**：[描述]
- **缺点来源**：[描述]
- **适用/不适用**：[描述]

## 8. 机会与建议

### 机会矩阵
[表格]

### 推荐路线
**路线1**: [名称]
- 组合：[派别组合]
- 理由：[说明]
- 风险：[说明]

### 创新切入点
1. **[切入点名称]**
   - 要改什么：[描述]
   - 为什么：[解决痛点]
   - 代价/风险：[描述]
   - 验证方式：[描述]

### 下一步行动
- [ ] [行动1]
- [ ] [行动2]
```

## 格式C: Notion 适配

### 页面结构建议

```
📁 [领域名称] 研究地图
├── 📄 核心论文数据库
├── 📄 派别对照表
├── 📄 时间线视图
├── 📄 深读计划
└── 📄 创新机会矩阵
```

### 数据库字段建议

**核心论文数据库**：
| 字段名 | 类型 | 说明 |
|--------|------|------|
| 论文标题 | Title | 论文标题 |
| 作者 | Text | 作者列表 |
| 年份 | Number | 发表年份 |
| 派别 | Select | 所属派别（选项） |
| 层级 | Select | 必读/选读/了解 |
| 相关度 | Number | 1-10评分 |
| 核心贡献 | Text | 一句话总结 |
| 关键假设 | Text | |
| 优点 | Multi-select | |
| 缺点 | Multi-select | |
| 阅读状态 | Select | 未读/进行中/已完成 |

**派别数据库**：
| 字段名 | 类型 | 说明 |
|--------|------|------|
| 派别名称 | Title | |
| 核心创意 | Text | |
| 边界定义 | Text | |
| 优点 | Text | |
| 缺点 | Text | |
| 适用场景 | Multi-select | |
| 不适用场景 | Multi-select | |
| 实施成本 | Select | 高/中/低 |

## 格式D: 飞书/多维表格适配

### 字段建议

**论文表**：
```
字段1: 论文标题 (文本)
字段2: 作者 (文本)
字段3: 年份 (数字)
字段4: 派别 (单选: 派别1/派别2/...)
字段5: 层级 (单选: 必读/选读/了解)
字段6: 相关度 (评分: 1-10)
字段7: 核心贡献 (多行文本)
字段8: 关键假设 (多行文本)
字段9: 阅读状态 (单选: 未读/进行中/已完成)
```

**派别表**：
```
字段1: 派别名称 (文本)
字段2: 核心创意 (多行文本)
字段3: 边界定义 (多行文本)
字段4: 优点 (多行文本)
字段5: 缺点 (多行文本)
字段6: 适用场景 (多选)
字段7: 实施成本 (单选: 高/中/低)
```

## 格式E: PPT 讲稿大纲

### 幻灯片规划

```markdown
# 幻灯片1: 标题页
[领域名称] 领域地图与研究机会

# 幻灯片2: 研究背景
- 研究问题定义
- 第一轮筛选结果概览
- [核心数据：论文数量/来源/时间跨度]

# 幻灯片3: 核心论文层级
- 必读论文：[3-5篇，含标题和核心贡献]
- 为什么这些是必读？

# 幻灯片4: 派别划分总览
- 识别出X个主要派别
- 每派一句话定义
- [可视化：派别关系图]

# 幻灯片5-7: 各派别详解
## 幻灯片5: 派别1
- 核心创意
- 代表论文
- 优点/缺点

## 幻灯片6: 派别2
...

## 幻灯片7: 派别3
...

# 幻灯片8: 时间线与演化
- 派别起源时间
- 关键演化节点
- 当前趋势

# 幻灯片9: 派别间批评关系
- 谁批评谁？
- 核心争议点
- [可视化：批评关系图]

# 幻灯片10: 核心收敛结果
- 最终20篇核心论文
- 选择的2-3个派别
- 选择理由

# 幻灯片11: 评价维度
- 领域关键问题
- 评价维度
- 场景权重差异

# 幻灯片12: 机会矩阵
- [表格：各派别的机会点]

# 幻灯片13: 推荐路线
- 路线1：[名称 + 理由]
- 路线2：[名称 + 理由]

# 幻灯片14: 创新切入点
- 切入点1：[描述]
- 切入点2：[描述]
- 切入点3：[描述]

# 幻灯片15: 下一步计划
- 必读论文顺序
- 需要补充检索的内容
- 要验证的关键假设

# 幻灯片16: Q&A
```

## 正文抽取模板

### 单篇论文抽取模板

```markdown
## [论文标题] - [作者 年]

### 3A 假设/成立条件
- **核心假设**：
  - [假设1]
  - [假设2]
- **现实可行性**：
  - [假设1]：[容易/困难满足，原因]
  - [假设2]：[容易/困难满足，原因]
- **违反后果**：
  - 如果[假设1]不成立：[后果]

### 3B 好处/贡献
1. **[贡献名称]**：[描述]
2. **[贡献名称]**：[描述]
3. 相比[之前工作]的改进：[具体描述]

### 3C 关键机制/公式
- **关键点1**：[描述]
- **关键点2**：[描述]
- **关键点3**：[公式/方法]
- **关键点4**：[描述]
- **关键点5**：[描述]

### 3D 缺点/局限
- **主要缺点**：
  - [缺点1]：[来源：理论/实现/数据]
  - [缺点2]：[来源]
- **局限性来源**：[分析]

### 适用性结论
- **适用场景**：[描述]
- **不适用场景**：[描述]

### Examples检查
- **成功案例**：[论文展示的场景]
- **潜在失败场景**（论文未提及）：[分析]
```

### 派别总结模板

```markdown
## [派别名称] 总结包

### 发展脉络
**起点**：[起源论文/年份]
**关键演化**：
- [年份1]：[节点描述]
- [年份2]：[节点描述]
**近期趋势**：[当前方向]

### 理论依据
- **数学/逻辑基础**：[描述]
- **核心理论支撑**：[引用理论/定理]

### 主要成果
- **代表性结果**：[描述]
- **典型应用**：[描述]

### 缺点来源
- **理论局限**：[描述]
- **实现瓶颈**：[描述]
- **数据/假设限制**：[描述]

### 适用/不适用结论
| 适用场景 | 不适用场景 |
|----------|------------|
| [场景1] | [场景2] |
| [场景3] | [场景4] |
```

