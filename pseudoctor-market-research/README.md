# Pseudoctor Market Research Analyst

专业市场调研分析师技能，融合严谨研究方法论、深度分析框架与专业报告标准。

---

## 快速开始

本技能用于生成50+页的专业市场研究报告，适用于投资决策、战略规划、竞争分析等场景。

### 核心特性

- **严谨研究方法论**：五阶段研究协议，确保数据准确性和可验证性
- **8章分析框架**：覆盖产品分析、市场环境、消费者洞察、渠道策略、投资建议等
- **专业报告标准**：符合顶级咨询公司的报告格式和质量要求
- **置信度标注**：所有数据标注来源和置信度，不确定因素明确标记
- **可视化系统**：6个核心图表 + 按需补充

---

## 使用方式

### 基本用法

```
用户：请帮我分析[主题/行业/公司]

分析师将：
1. 确认研究需求
2. 执行五阶段研究协议
3. 应用8章分析框架
4. 生成包含6个核心可视化的50+页报告
```

### 示例场景

#### 场景1：行业分析
```
用户：请分析中国成人奶粉市场

分析师输出：
- 50+页行业分析报告
- 重点：市场规模、竞争格局、渠道策略、投资价值
- 包含：TAM/SAM/SOM分析、Porter五力、SWOT等
```

#### 场景2：公司研究
```
用户：从投资角度分析飞鹤公司

分析师输出：
- 30-40页投资研究报告
- 重点：投资逻辑、估值前景、风险评估、投资评级
- 包含：财务分析、竞争优势、增长潜力
```

#### 场景3：聚焦分析
```
用户：重点分析飞鹤的渠道策略

分析师输出：
- 15-20页聚焦分析报告
- 重点：渠道结构、下沉市场策略、管理体系
- 包含：渠道效率评估、优化建议
```

## 分析框架概览

### 第一部分：执行摘要

市场快照 + 核心发现 + 战略建议

### 第二部分：核心产品与产品线分析

产品定义 + 产品线矩阵 + 竞争力评估 + 生命周期

### 第三部分：市场宏观环境与现状

市场规模（TAM/SAM/SOM） + PESTEL分析 + 竞争格局（含Porter五力）

### 第四部分：目标市场表现与消费者洞察

市场地位 + 消费者画像 + 购买决策旅程

### 第五部分：销售渠道与分销体系剖析（重点）

渠道全景 + 线下渠道深度剖析 + 效率评估

### 第六部分：未来发展趋势与潜力预测

趋势预判 + SWOT分析 + 潜力与风险

### 第七部分：战略建议

总体战略 + 产品策略 + 营销策略 + 渠道策略 + 实施路径

### 第八部分：投资研究建议

投资逻辑 + 估值前景 + 风险评估 + 投资评级 + 免责声明

## 研究方法论

### 五阶段协议

1. **PHASE 1: 查询验证** - 明确研究需求
2. **PHASE 2: 深度分析** - 分解问题、生成假设
3. **PHASE 3: 系统研究** - 多轮搜索、来源评估
4. **PHASE 4: 协作增强** - 识别缺口、请求协助
5. **PHASE 5: 综合输出** - 整合信息、标注置信度

### 来源评估体系

- **Tier 1**（⭐⭐⭐⭐⭐）：官方、学术、政府
- **Tier 2**（⭐⭐⭐⭐）：媒体、专家、咨询
- **Tier 3**（⭐⭐⭐）：多源印证
- **Reject**：无来源、被证伪、偏见

### 置信度标注

- **95%+**：多个权威来源一致
- **85-94%**：多个可信来源一致
- **70-84%**：单一权威或多个部分一致
- **50-69%**：单一次级或信息有限
- **<50%**：信息不足，仅供参考

## 可视化系统

### 6个核心图表

1. **执行摘要信息图** - 核心数据概览
2. **市场增长轨迹图** - 历史+预测
3. **TAM/SAM/SOM同心圆图** - 市场空间层级
4. **Porter五力模型图** - 行业吸引力
5. **竞争定位矩阵** - 竞争格局位置
6. **风险热力图** - 风险优先级

- `scientific-schematics`：专业图表
- `generate-image`：信息图

## 输出格式

### 主要格式

- **LaTeX（PDF）**：专业报告，用于正式交付
- **Markdown**：便于编辑和转换Word

### 页数标准

- **最小**：32页
- **目标**：43-58页
- **可扩展**：根据需要可超过60页

---

## 质量标准

### 数据质量

- 核心数据：≥3个Tier 1来源
- 重要论断：≥2个独立来源
- 时效性：≤2年
- 置信度：明确标注

### 逻辑质量

- 分析框架正确应用
- 推理链条完整
- 前后逻辑一致
- 无自相矛盾

### 格式质量

- 专业术语准确
- 无冗余表述
- 风格统一
- 结构清晰
