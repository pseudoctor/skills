# 分阶段确认工作流程 (Staged Confirmation Workflow)

> **目的**: 避免一次性生成不符合要求的报告，通过分阶段确认提高成功率

---

## 工作流程概述

```
需求对齐 → 大纲确认 → 内容初稿 → 完整报告 → 质量检查
   ↓          ↓          ↓          ↓          ↓
 Phase 0    Phase 1    Phase 2    Phase 3    Phase 4
```

---

## Phase 0: 需求对齐（新增）

### 目的
确保理解用户需求，明确研究范围和输出要求

### 执行步骤

#### 0.1 需求收集

向用户确认：

```
📋 研究需求确认

【基础信息】
1. 研究主题：[用户输入]
2. 研究目的：[投资决策/战略规划/竞争分析/市场进入/其他]
3. 目标读者：[内部管理层/外部投资者/合作伙伴/其他]

【范围界定】
4. 地理范围：[全球/中国/特定区域]
5. 时间范围：[历史年数/预测年数]
6. 细分市场：[整体市场/特定细分]

【深度要求】
7. 报告深度：[标准/深度/综合]
8. 重点关注：[产品/渠道/竞争/财务/技术/其他]
9. 特殊要求：[如有]

【参考文档】（重要！）
10. 是否有参考报告？[是/否]
    如有，请提供路径，我将分析其内容密度和风格
```

#### 0.2 需求确认响应

根据用户输入，生成需求确认书：

```
✅ 需求确认书

【研究主题】飞鹤成人奶粉市场深度分析

- 地理范围：中国大陆
- 时间范围：2019-2024（历史），2025-2030（预测）
- 细分市场：功能性成人奶粉（50+人群为主）

【输出规格】
- 报告类型：深度战略研究报告
- 预计篇幅：50-60页
- 表格数量：35-40个
- 数据点：200+个
- 主要章节：8章

【时间规划】
- 需求对齐：已完成
- 大纲生成：Phase 1
- 内容撰写：Phase 2-3
- 质量检查：Phase 4
- 预计完成：[时间估算]

【风险提示】
- 部分最新数据可能需要付费数据库
- 竞品详细数据可能存在信息不对称

✋ 请确认以上理解是否准确？
回复"确认"继续，或提出修改建议
```

## Phase 1: 大纲生成与确认

### 目的
在生成内容前，先让用户确认报告结构，避免方向错误

### 执行步骤

#### 1.1 生成大纲

根据需求和参考文档（如有），生成详细大纲：

```
📋 报告大纲（待确认）

## 总体规模
- 预计页数：50-60页
- 表格数量：35-40个
- 数据点：200+个
- 主要章节：8章

## 章节结构

### 执行摘要（3页）
✓ 市场快照表格（5-8个关键指标）
✓ 核心发现清单（5条，每条有数据支撑）
✓ 战略建议清单（5条，按优先级）
✓ 投资评级：买入/持有/卖出

### 第一章：核心产品与产品线分析（5页）
1.1 核心产品定义
  → 产品功能特性表
1.2 产品线矩阵分析
  → 细分市场覆盖表（5个细分×产品线）
1.3 产品竞争力评估
  → USP对比表（飞鹤 vs 竞品，4维度）
1.4 产品生命周期
  → 生命周期阶段表（产品×阶段×收入占比）

**预计数据点：** 30个
**预计表格：** 5个

### 第二章：市场宏观环境与现状（10页）⭐重点
2.1 市场规模与增长
  → 历史数据表（2019-2024，6年）
  → TAM/SAM/SOM分层表
  → 预测数据表（2025-2030，5年）
2.2 PESTLE分析
  → PESTLE分析矩阵（6维度×3因素）
2.3 竞争格局
  → 竞争对手份额表（8家，3年）
  → Porter五力评分表
  → 市场集中度表（CR5/CR10/HHI）

**预计数据点：** 80个
**预计表格：** 10个

### 第三章：目标市场表现与消费者洞察（5页）
3.1 市场地位与份额
  → 市场份额变化表（3年）
3.2 消费者画像
  → 消费者分群表（3个群体×特征）
3.3 购买决策旅程
  → 决策旅程表（4阶段×渠道×转化率）
3.4 痛点与需求
  → 痛点优先级矩阵（10个痛点）

**预计数据点：** 35个
**预计表格：** 5个

### 第四章：销售渠道与分销体系（8页）⭐重点
4.1 渠道全景图
  → 全渠道列表（线上×线下，20+渠道）
4.2 城市分级渗透
  → 渗透率表（一二三四线城市）
4.3 渠道效率分析
  → 效率指标表（覆盖率/动销率/坪效）
4.4 竞品渠道对比
  → 渠道对比表（飞鹤 vs 3家竞品）

**预计数据点：** 50个
**预计表格：** 8个

### 第五章：未来发展趋势与潜力（5页）
5.1 关键趋势分析
  → 趋势影响矩阵（10个趋势）
5.2 SWOT分析
  → SWOT分析表（4维度×5项）
5.3 增长机会识别
  → 机会排序表（市场规模×概率×优势）

**预计数据点：** 30个
**预计表格：** 4个

### 第六章：战略建议（5页）
6.1 总体战略定位
6.2 产品策略建议
6.3 市场与营销策略
6.4 渠道策略建议
  → 实施路线图（阶段×动作×时间）

**预计数据点：** 20个
**预计表格：** 3个

### 第七章：投资研究建议（4页）
7.1 投资逻辑
  → 投资逻辑表（论点×证据×置信度）
7.2 估值分析
  → 估值对比表（历史/同行/目标）
7.3 风险提示
  → 风险清单表（5个风险×应对）
7.4 投资评级
  → 评级表（定性+定量+理由）

**预计数据点：** 25个
**预计表格：** 4个

### 附录（5页）
- A: 数据来源详细列表
- B: 研究方法论说明
- C: 竞争对手详细档案
- D: 术语表
```

## Phase 2: 内容初稿（分章节生成）

### 目的
逐章生成内容，每章完成后请用户确认，避免整体方向错误

### 执行步骤

#### 2.1 章节生成顺序

按以下顺序生成：

```
执行摘要 → 第二章 → 第四章 → 第一章 → 第三章
         → 第五章 → 第六章 → 第七章 → 附录
```

**顺序说明：**
1. 先执行摘要：让用户快速了解整体方向
2. 再核心章节（第2、4章）：最重要的部分优先确认
3. 然后其他章节：按重要性依次生成
4. 最后附录：汇总整理

#### 2.2 单章生成流程

每生成一章，执行：

```
【生成中】第一章：核心产品与产品线分析

📊 本章统计：
- 预计页数：5页
- 预计表格：5个
- 预计数据点：30个

✅ 已完成内容：
1.1 核心产品定义 ✓
  - 产品功能特性表 ✓
1.2 产品线矩阵分析 ✓
  - 细分市场覆盖表 ✓
1.3 产品竞争力评估 ✓
  - USP对比表（飞鹤 vs 伊利/蒙牛） ✓
1.4 产品生命周期 ✓
  - 生命周期阶段表 ✓

📋 本章数据点：30个
📋 本章表格：5个

📄 预览（关键发现）：
- 飞鹤成人奶粉产品线覆盖3大细分市场
- "爱本"系列定位高端，价格带100-300元
- 核心竞争优势：全产业链+科技壁垒（乳铁蛋白）
- 与竞品对比：价格高于伊利/蒙牛，技术投入领先

---

✋ 请确认本章内容

回复选项：
1. "确认" - 继续下一章
2. "扩展XXX" - 扩展某小节
3. "调整XXX" - 修改某内容
4. "重新生成" - 本章重新写
```

#### 2.3 用户反馈处理

**确认（"确认"）**
- 继续生成下一章

**扩展（"扩展1.2节"）**
- 扩展指定小节，增加更多数据和分析
- 重新展示供确认

**调整（"调整1.3节，增加XX竞品"）**
- 根据调整意见修改
- 重新展示供确认

**重新生成（"重新生成"）**
- 本章完全重新撰写
- 分析之前版本的问题，避免重复

|--------|---------------------|---------------------|
| 成功率 | 30% | 90% |

| 用户参与度 | 仅开始和结束 | 全程参与 |

| 用户满意度 | 需要多次调整 | 一次性通过 |
| 时间成本 | 长（多次返工） | 短（一次通过） |
