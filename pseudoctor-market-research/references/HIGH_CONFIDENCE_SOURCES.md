# 高置信度信息源指南 (High-Confidence Sources Guide)

> **目的**: 确保市场研究报告中使用的信息来源可信、准确、可验证

---

## 核心原则

### 优先级排序
```
Tier 1 (最高置信度) > Tier 2 > Tier 3 > 拒绝使用
```

**每次研究都必须：**
1. 使用网络搜索获取最新数据
2. 优先选择Tier 1来源
3. 交叉验证关键数据（≥2个独立来源）
4. 明确标注置信度

---

- [ ] 预测数据是否有明确时间基准？

- [ ] 是否有明显偏向？
- [ ] 资助方是谁？

## 交叉验证方法

### 关键数据必须交叉验证

对于以下类型的数据，必须≥2个独立来源：

**必须验证的数据**:
- 市场规模
- 市场份额
- 增长率
- 公司财务数据
- 关键统计指标

**验证方法**:

```bash
# 示例：验证成人奶粉市场规模

# 来源1
site:euromonitor.com China adult milk powder market size 2024

# 来源2
site:iresearch.com.cn 成人奶粉 市场规模 2024

# 来源3（如果前两个不一致）
中国 乳制品 行业报告 2024 filetype:pdf

# 对比三个来源的数据
# 如果差异<10%，取平均值
# 如果差异>10%，说明原因并标注置信度降低
```

## 置信度标注规范

### 根据来源数量标注置信度

| 来源情况 | 置信度 | 标注格式 |
|---------|--------|----------|
| 3+个Tier 1来源一致 | 95%+ | `[置信度: 95%]` |
| 2-3个Tier 1来源基本一致 | 85-94% | `[置信度: 90%]` |
| 1个Tier 1来源 | 70-84% | `[置信度: 75%]` |
| 1-2个Tier 2来源 | 50-69% | `[置信度: 60%]` |
| 单一次级来源或信息不足 | <50% | `[置信度: 40%]` |

### 标注示例

```markdown
中国成人奶粉市场规模达到286.82亿元
[来源: 欧睿咨询, 2024; 艾瑞咨询, 2024; 中国乳制品工业协会]
[置信度: 90%]

飞乳成人粉市场份额为8-10%
[来源: 欧睿咨询, 2024]
[置信度: 75%] - 单一来源，需进一步验证

预计2030年市场规模达到500亿元
[基于: 人口老龄化加速假设; CAGR 12%]
[置信度: 65%] - 预测数据，依赖假设
```

## 实际操作示例

### 案例：飞鹤成人奶粉市场分析

#### Step 1: 基础信息收集

```bash
# Google搜索
"飞鹤成人奶粉" 市场份额 2024
feihe adult milk powder market share

# Bing搜索
飞鹤乳业 成人奶粉 业务 2024
```

**结果示例**:
- 发现飞鹤成人粉业务快速增长
- 识别主要竞品：伊利欣活、蒙牛悠瑞
- 了解"爱本"品牌定位

---

#### Step 2: 权威数据搜索

```bash
# 市场规模
site:euromonitor.com China adult milk powder 2024
site:iresearch.com.cn 成人奶粉 市场规模

# 公司数据
飞鹤乳业 年报 2024 filetype:pdf
site:feihe.com 投资者关系

# 竞争格局
中国 成人奶粉 品牌 市场份额 排名
```

**结果示例**:
- 欧睿: 成人奶粉市场286.82亿元
- 飞鹤年报: 其他乳制品15.14亿元
- 艾瑞: 飞鹤成人粉份额8-10%

---

#### Step 3: 交叉验证

```bash
# 验证市场规模
欧睿 成人奶粉 286.82亿 vs
艾瑞 成人奶粉 270-300亿 vs
行业报告 成人奶粉 约290亿

# 结论: 数据基本一致，取286.82亿元
# 置信度: 90%
```

---

#### Step 4: 最新动态

```bash
# 2024年最新
飞鹤乳业 2024 年报 filetype:pdf
飞鹤 成人奶粉 2024 业绩

# 行业政策
乳制品 行业 政策 2024 最新
```

**结果示例**:
- 2024年报: 其他乳制品收入15.14亿元，增长6.3%
- 行业政策: 支持银发经济，利好成人营养品
