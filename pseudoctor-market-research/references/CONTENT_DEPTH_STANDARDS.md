# 内容深度标准 (Content Depth Standards)

> **新增章节 - 插入SKILL.md第916行（页数分配目标之后）**

---

## 内容深度标准

### 总体要求

每页报告必须包含：
- **≥4个数据点**：具体数值、百分比、金额等
- **≥1个来源引用**：标注数据来源
- **≥1个分析洞察**：不只是数据罗列，要有分析

### 分章节内容标准

#### 执行摘要（2-3页）

**必须包含：**
- [ ] 市场快照表格（5-8个关键指标）
- [ ] 核心发现清单（3-5条，每条有数据支撑）
- [ ] 战略建议清单（3-5条，按优先级排序）
- [ ] 投资评级（如适用）

**数据要求：**
- 表格：≥1个，5-8行×3-4列
- 数据点：≥20个
- 来源：≥5个不同来源

---

#### 第一章：核心产品与产品线分析（4-5页）

**必须包含的表格：**
- [ ] 产品定义表（功能/特性/目标客户）
- [ ] 产品线矩阵表（细分市场×产品覆盖度）
- [ ] 竞争力评估表（USP/技术/成本/品牌，至少4维度）
- [ ] 产品生命周期表（阶段×产品×收入占比）

**必须包含的数据点：**
- 各产品线营收占比（≥3年数据）
- 各产品线毛利率对比
- 核心技术指标（如适用）
- 与竞品对比数据（≥3家）

**分析深度：**
- 不只是产品介绍，要分析：
  - 为什么这个产品能成功/失败
  - 产品之间的协同/冲突关系
  - 产品线的健康度评分

---

#### 第二章：市场宏观环境与现状（6-8页）⭐最重要

**必须包含的表格：**
- [ ] 历史市场规模表（2019-2024，至少6年数据）
- [ ] 市场规模预测表（2025-2030，至少5年预测）
- [ ] TAM/SAM/SOM分层表（具体数值+测算依据+来源）
- [ ] PESTLE分析表（每个维度≥3个因素，带影响评级）
- [ ] 竞争对手市场份额表（至少8家竞争对手，≥3年数据）
- [ ] Porter五力评分表（每个力1-5分+理由）

**必须包含的数据点：**
- ≥20个具体数值（带来源和置信度）
- ≥5个不同数据源的交叉验证
- ≥3个CAGR计算（历史+预测）
- 市场集中度指标（CR5/CR10/HHI）

**分析框架：**
- TAM/SAM/SOM必须详细说明：
  - TAM是如何定义的？
  - SAM的筛选标准是什么？
  - SOM的获取假设是什么？
- Porter五力必须评分：
  - 每个力1-5分
  - 给出具体理由
  - 综合评估行业吸引力

**示例表格格式：**

```
表2-1：TAM/SAM/SOM市场空间测算

| 层级 | 定义 | 市场规模(亿元) | 测算依据 | 数据来源 |
|------|------|---------------|----------|----------|
| TAM | 中国成人奶粉市场 | 450.2 | 全年龄段×人均消费 | 欧睿[置信度:90%] |
| SAM | 功能性成人奶粉 | 286.8 | 50+人群×高端消费 | 欧睿+国家统计局[置信度:85%] |
| SOM | 飞鹤可获得市场 | 30.5 | SAM×12%市占率目标 | 公司测算[置信度:60%] |
```

---

#### 第三章：目标市场表现与消费者洞察（4-5页）

**必须包含的表格：**
- [ ] 消费者画像表（人口统计×心理特征×行为特征，至少3个细分群体）
- [ ] 购买决策旅程表（4阶段×渠道×触点×转化率）
- [ ] 痛点优先级矩阵（重要性×紧迫性，至少10个痛点）
- [ ] 品牌健康度指标表（知名度/美誉度/忠诚度，对比竞品）

**必须包含的数据点：**
- 各细分群体规模（人数×消费能力）
- 各渠道的渗透率和转化率
- 痛点发生频率（百分比）
- 品牌指标具体数值

**分析深度：**
- 不只描述消费者"是什么"，要分析"为什么"
- 例如：为什么50+群体更喜欢线下购买？
  - 原因1：信任问题（数据支撑）
  - 原因2：使用习惯（调研数据）
  - 原因3：社交需求（行为分析）

---

#### 第四章：销售渠道与分销体系剖析（6-8页）⭐重点

**必须包含的表格：**
- [ ] 渠道全景图（线上×线下，包含所有子渠道）
- [ ] 各渠道渗透率表（一二三四线城市，至少20个渠道）
- [ ] 渠道效率指标表（覆盖率/动销率/坪效/费用率）
- [ ] 经销商分级体系表（一级/二级/三级，数量+覆盖）
- [ ] 渠道成本结构表（各环节利润分配）
- [ ] 竞品渠道对比表（至少3家竞品）

**必须包含的数据点：**
- 各层级城市渠道覆盖率（百分比）
- 各渠道的平均单店产出
- 渠道成本占比（对比竞品）
- 经销商数量（分级统计）

**分析深度：**
- 不只罗列渠道，要分析：
  - 为什么某些渠道表现好/差？
  - 渠道冲突在哪里？如何解决？
  - 下沉市场的特殊挑战是什么？

---

#### 第五章：未来发展趋势与潜力预测（4-5页）

**必须包含的表格：**
- [ ] 趋势影响矩阵（趋势×影响度×可能性，至少10个趋势）
- [ ] SWOT分析表（每个维度≥5项，带重要性评分）
- [ ] 增长机会排序表（市场规模×增长概率×飞鹤优势）
- [ ] 风险清单表（风险×概率×影响×应对措施）

**必须包含的数据点：**
- 各趋势的量化影响（如市场规模变化）
- SWOT各要素的权重评分
- 机会的优先级评分
- 风险的概率和影响评分

**分析深度：**
- 每个趋势要有：
  - 具体的时间预测（何时发生？）
  - 量化影响（市场规模变化多少？）
  - 应对建议（如何把握/规避？）

---

#### 第六章：战略建议（4-5页）

**必须包含的表格：**
- [ ] 战略选项评估表（选项×投资×回报×风险×优先级）
- [ ] 实施路线图（阶段×关键动作×时间×责任人×里程碑）
- [ ] 资源配置建议表（项目×资金×人力×时间）

**必须包含的建议：**
- ≥5条具体战略建议
- 每条建议包含：
  - 具体行动项（做什么？）
  - 预期效果（达到什么目标？）
  - 所需资源（需要多少钱/人？）
  - 实施时间（何时完成？）
  - 风险应对（可能出现什么问题？如何解决？）

---

#### 第七部分：投资研究建议（3-4页）

**必须包含的表格：**
- [ ] 投资逻辑表（论点×证据×置信度）
- [ ] 估值对比表（历史估值/同行估值/目标估值）
- [ ] 敏感性分析表（不同情景下的估值）
- [ ] 投资评级表（定性评级+定量目标+核心理由）

**必须包含的分析：**
- 投资逻辑：≥3个核心理由，每个有数据支撑
- 估值分析：多种方法（PE/PS/DCF等）交叉验证
- 风险提示：≥5个关键风险，每个有应对建议
- 投资评级：明确的买入/持有/卖出建议

### 数据表格标准

#### 表格数量要求

| 章节 | 最少表格数 | 推荐表格数 |
|------|-----------|-----------|
| 执行摘要 | 1 | 1-2 |
| 第一章 | 4 | 5-7 |
| 第二章 | 6 | 8-10 |
| 第三章 | 4 | 5-7 |
| 第四章 | 6 | 8-10 |
| 第五章 | 4 | 5-7 |
| 第六章 | 3 | 4-6 |
| 第七部分 | 4 | 5-7 |
| **总计** | **32** | **42-56** |

#### 表格质量标准

每个表格必须：
1. **有标题**：如 "表2-1：中国成人奶粉市场规模（2019-2024）"
2. **有单位**：所有数值列标注单位（亿元/%/万人等）
3. **有来源**：标注数据来源
4. **有置信度**：预测数据标注置信度
5. **有分析**：表格后有简要分析（2-3句话）

#### 表格格式示例

```
表2-1：中国成人奶粉市场规模与增长（2019-2024E）

| 年份 | 市场规模(亿元) | 同比增长率 | 50+人群贡献 | 数据来源 |
|------|---------------|-----------|------------|----------|
| 2019 | 152.3 | - | 52% | 欧睿[置信度:95%] |
| 2020 | 168.5 | +10.6% | 55% | 欧睿[置信度:95%] |
| 2021 | 192.7 | +14.4% | 58% | 欧睿[置信度:95%] |
| 2022 | 215.3 | +11.7% | 60% | 欧睿[置信度:95%] |
| 2023 | 241.2 | +12.0% | 61% | 欧睿[置信度:95%] |
| 2024E | 286.8 | +18.9% | 62% | 预测[置信度:75%] |

数据来源：欧睿咨询、国家统计局、公司年报
CAGR（2019-2024E）：13.5%

分析要点：
1. 市场保持稳定增长，CAGR达13.5%，主要受人口老龄化驱动
2. 50+人群贡献率持续提升，从2019年的52%增至2024E的62%
3. 2024年增速加快至18.9%，源于银发经济觉醒和产品高端化
```

### 数据点密度标准

#### 每页数据点要求

| 页面类型 | 最少数据点 | 推荐数据点 |
|---------|-----------|-----------|
| 文字分析页 | 4 | 6-8 |
| 表格页 | 10 | 15-20 |
| 图表页 | 5 | 8-10 |
| 综合页 | 6 | 8-12 |

#### 数据点类型分配

每页应包含多种类型的数据点：
- **市场规模**：具体金额、增长率、占比
- **时间数据**：年份、季度、预测时间
- **对比数据**：同比、环比、竞品对比
- **排名数据**：市场份额排名、增长率排名
- **评估数据**：评分（1-5分）、评级（买入/持有）

### 置信度标注规范

#### 置信度等级

| 置信度 | 说明 | 使用场景 |
|--------|------|----------|
| 95%+ | 多个权威来源一致确认 | 历史数据、官方统计 |
| 85-94% | 多个可信来源基本一致 | 行业报告、公司财报 |
| 70-84% | 单一权威来源 | 权威机构预测 |
| 50-69% | 单一次级来源 | 一般媒体报道 |
| <50% | 信息不足 | 粗略估算 |

#### 标注格式

```markdown
**数据** [置信度: XX%]
**数据** 来源: [机构名称] [置信度: XX%]
**数据** (基于XX假设) [置信度: XX%]
```

示例：
- "市场规模286.8亿元 [置信度: 75%]"
- "CAGR 13.5% 来源: 欧睿咨询 [置信度: 90%]"
- "2030年预计达到500亿元 (基于人口老龄化加速假设) [置信度: 65%]"

### 来源引用规范

#### Tier 1 来源（首选）

- 官方机构：国家统计局、政府部门
- 学术期刊：同行评议论文
- 权威咨询：Gartner、Forrester、IDC、欧睿
- 公司财报：年报、季报
- 行业协会：官方统计数据

#### Tier 2 来源（辅证）

- 知名媒体：财经媒体报道
- 行业专家：公开演讲、访谈
- 分析师报告：券商研报
- 调研机构：消费者调研

#### Tier 3 来源（参考）

- 多个相互印证的次级来源
- 一般媒体报道
- 公司公告（非财报）

#### 引用格式

```markdown
数据 [来源: 机构名称]
数据 [来源: 机构名称, 报告名称, 发布日期]
数据 (来源1; 来源2; 来源3) [交叉验证]
```

### 质量检查清单

#### 章节完成度检查

每完成一章，检查：

**第二章检查清单：**
- [ ] 历史数据≥6年
- [ ] 预测数据≥5年
- [ ] TAM/SAM/SOM表格完整
- [ ] Porter五力有评分和理由
- [ ] PESTLE每个维度≥3因素
- [ ] 竞争对手≥8家
- [ ] 所有数据有来源
- [ ] 预测数据有置信度
- [ ] 分析不只数据罗列
- [ ] 表格后有分析要点

**第四章检查清单：**
- [ ] 渠道全景图完整
- [ ] 城市分级覆盖（一二三四线）
- [ ] 效率指标齐全
- [ ] 竞品对比≥3家

#### 整体质量检查

报告完成后，检查：

**数量检查：**
- [ ] 总页数≥50页
- [ ] 表格总数≥35个
- [ ] 数据点总数≥200个
- [ ] 不同来源≥10个

**质量检查：**
- [ ] 所有预测有置信度
- [ ] 所有数据有来源
- [ ] 核心数据有多源验证
- [ ] 分析有洞察（不只是描述）
- [ ] 建议可操作（有具体行动项）

## 示例：第二章完整内容结构

```
第二章：市场宏观环境与现状

2.1 市场规模与增长分析
  2.1.1 当前市场容量
    → 表2-1：历史市场规模（2019-2024）
    → 数据：6年×4个指标=24个数据点
    → 分析：增长趋势、驱动因素

  2.1.2 TAM/SAM/SOM分析
    → 表2-2：TAM/SAM/SOM测算
    → 图2-1：TAM/SAM/SOM同心圆图
    → 分析：三层市场定义和获取策略

  2.1.3 未来市场预测
    → 表2-3：市场规模预测（2025-2030）
    → 数据：6年×3个情景=18个数据点
    → 分析：基准/乐观/悲观情景

2.2 宏观环境分析（PESTLE）
  → 表2-4：PESTLE分析矩阵
  → 6个维度×3个因素=18个因素
  → 每个因素有影响评级和理由

2.3 竞争格局分析
  2.3.1 市场份额
    → 表2-5：竞争对手市场份额（≥8家，3年）
    → 数据：8家×3年=24个数据点

  2.3.2 Porter五力分析
    → 表2-6：Porter五力评分
    → 图2-2：Porter五力图
    → 5个力×评分×理由

  2.3.3 市场集中度
    → 表2-7：市场集中度指标（CR5/CR10/HHI）
    → 分析：行业结构判定

第二章总计：
- 页数：7-10页
- 表格：7个
- 图表：2个
- 数据点：100+个
```
