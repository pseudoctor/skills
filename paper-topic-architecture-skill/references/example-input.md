# 示例输入

下面给出 3 组最小可复制输入，分别对应三种模式。可直接粘贴给 Codex、Claude Code、Gemini CLI 使用。

## Mode A：从研究兴趣出发

```text
请使用 $paper-topic-architecture-master。

我的研究主题是：中国特色大国外交中的正确义利观。
学科方向：国际关系 / 马克思主义理论。
我目前有一批筛选后的文献和一个初步领域地图，但还没有正式题目。
我希望更偏：机制分析 + 场景比较。
希望避免的方向：纯口号式阐释、泛泛政策解读。
目标：先帮我生成 3-5 个有核心期刊潜力的题目，并说明各自的研究路径差异。
```

## Mode B：给定题目 + 大纲优化

```text
请使用 $paper-topic-architecture-master。

我已有题目：
《从规范理念到外交实践：正确义利观的作用机制及其应用边界》

我已有初步大纲：
一、引言
二、正确义利观的理论基础
三、正确义利观在中国外交中的体现
四、正确义利观的现实意义
五、结论

请不要从头重新拟题，先按 skill 的优化模式做：
1. 题目诊断
2. 大纲诊断
3. 给出 2-3 套修订方案
4. 推荐一个更适合核心期刊的版本
```

## Mode C：近完成版大纲诊断

```text
请使用 $paper-topic-architecture-master。

我已经有较完整的大纲和文献综述框架，想按中文核心期刊标准查漏补缺。

请重点检查：
- 问题意识是否清楚
- 文献综述是否真正导向本文切口
- 第四部分分析是否够强
- 讨论与结论是否有重复
- 贡献表达是否成立

如果发现问题，请优先给我：
1. 结构性缺陷
2. 最小修改方案
3. 如需重构，再给重构版
```
